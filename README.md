## Manual Config(手动配置)

#### php
修改php.ini中的extension_dir配置，改为php目录下ext目录的绝对路径

#### apache

修改conf目录下httpd.conf中的LoadModule php7_module的路径，改为php目录下php7apache2_4.dll文件的绝对路径

#### mysql

修改my.ini文件中base_dir和data_dir的配置，分别修改为mysql目录和mysql目录下的data目录的绝对路径
